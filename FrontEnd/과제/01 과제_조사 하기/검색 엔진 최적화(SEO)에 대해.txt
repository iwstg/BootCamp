# 검색 엔진 최적화 (SEO: Search Engine Optimization)

### SEO 란?
SEO란 웹사이트가 검색 엔진(Google, Naver, Bing 등)의 검색 결과에서 더 상위에 노출되도록 최적화하는 작업을 말합니다.
목적: 

### 목적

• 검색 사용자가 웹사이트를 더 쉽게 찾도록 하여 **트래픽(방문자 수)**을 늘리는 것.

### 왜 SEO가 중요한가?
|이유|설명|
|:--:|--|
|높은 노출 효과|검색 결과 상위에 나타날수록 클릭률 증가|
|광고비 절약|유료 광고 없이도 자연 검색에서 유입 가능|
|브랜드 신뢰도 향상|상위에 노출된 사이트는 더 신뢰받는 경향|
|지속적 트래픽 확보|광고 예산 없이도 꾸준한 방문 가능|

### 주요 SEO 요소

1. 온페이지 SEO (On-page SEO)
>웹사이트 내부에서 설정하는 최적화 요소들
>
| 항목                   | 설명                                 |
| -------------------- | ---------------------------------- |
| `<title>` 태그         | 브라우저 탭 제목 및 검색 결과 제목으로 표시됨         |
| `<meta description>` | 검색 결과에 표시되는 요약문                    |
| `<h1>`, `<h2>` 등     | 제목 구조가 잘 짜여 있을수록 검색 엔진이 이해하기 쉬움    |
| 키워드 최적화              | 페이지 주제와 관련된 검색어를 적절히 포함            |
| 내부 링크                | 페이지 간 연결을 통해 검색 엔진이 구조를 잘 이해하게 함   |
| 이미지 대체 텍스트 (`alt`)   | 시각 장애인을 위한 접근성과 함께 이미지도 검색 가능하게 만듦 |


2. 오프페이지 SEO (Off-page SEO)
>웹사이트 외부에서 이루어지는 최적화
>
| 항목            | 설명                                  |
| ------------- | ----------------------------------- |
| 백링크(Backlink) | 다른 사이트에서 내 사이트로 연결된 링크 수와 품질        |
| SNS 공유        | 소셜 미디어에서 자주 언급되는 콘텐츠일수록 검색엔진 신뢰도 상승 |
| 브랜드 언급        | 온라인 상에서 브랜드 이름이 언급될수록 신뢰도 상승        |
	

3. 기술적 SEO (Technical SEO)
>검색 엔진이 사이트를 잘 읽고 이해할 수 있도록 돕는 기술적 요소
>
| 항목         | 설명                     |
| ---------- | ---------------------- |
| 사이트 속도     | 빠를수록 사용자 경험 및 SEO에 유리  |
| 모바일 최적화    | 반응형 웹 디자인은 필수          |
| 사이트맵       | 검색 엔진에 페이지 구조를 알려주는 파일 |
| robots.txt | 검색엔진이 크롤링할 수 있는 범위를 지정 |
